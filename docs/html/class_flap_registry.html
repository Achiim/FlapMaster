◆ availabilityCheck()

void FlapRegistry::availabilityCheck ( )

Will be called cyclic by Registry Task with the help of a countdown timer and will trigger the twins of all registered devices to ping them and:

if device answers:

  • ask device about bootFlag
  • reset bootFlag
  • calibrate device

if device does not answer:

  • de-register this device
